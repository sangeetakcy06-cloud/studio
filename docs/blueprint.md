# **App Name**: Career Navigator

## Core Features:

- Skill Extraction: AI-powered extraction of skills from resumes, LinkedIn profiles, and GitHub portfolios using NLP models (Spacy, BERT).
- Job Matching: Matching extracted skills with open roles from job APIs (LinkedIn, Internshala, RapidAPI).
- Personalized Dashboard: Display a personalized dashboard showing recommended jobs, internships, and hackathons, along with skill gaps identified.
- Skill Gap Analysis Tool: AI-driven identification of missing skills based on job market demands, and recommendations for relevant learning resources to fill those gaps. The LLM will use reasoning as a tool to make its determination.
- Portfolio Upload: Allow users to upload their resume or link their LinkedIn and GitHub profiles for automated skill extraction.
- Job Application Tracking: Enable users to track the status of their job applications within the platform.
- Data Storage: Store user data, extracted skills, and job application history in Firestore.

## Style Guidelines:

- Primary color: Slate blue (#708090) to convey professionalism and reliability.
- Background color: Very light gray (#F0F0F0), a desaturated version of slate blue, for a clean and modern look.
- Accent color: Soft teal (#66CDAA) to highlight key actions and information.
- Maintain a clean and structured layout with clear sections for job recommendations, skill gaps, and application tracking.
- Body and headline font: 'Inter' (sans-serif) for a modern, neutral, and objective feel.
- Use consistent, professional icons from a library like Material Design Icons to represent job roles, skills, and industries.
- The logo should be abstract, representing growth and direction. It could incorporate upward-pointing arrows or stylized compass imagery in the slate blue and soft teal color palette.
- Incorporate subtle transitions and animations to enhance user engagement and provide feedback on interactions, like loading new recommendations.
